# Knockoff
Knockoff is a Hack Club YSWS website built with Reflex.

The premise is straightforward: pick a closed-source product, build a usable open-source alternative, ship it, and earn points for rewards that help you keep building.

## The YSWS
Knockoff is a YSWS about rebuilding closed products in the open.

The loop is:
- you pick a closed-source app
- you build a real open-source version of its core workflow
- you make it self-hostable and document it properly
- you earn points from coding time and bonus tasks
- you spend those points in the Knockoff shop

## Running locally
This project uses `uv` for dependency management.

```bash
uv sync
uv run reflex run
```
